# ETF 基金分析方法详细文档

本文档详细描述了 ETF Challenger 系统中使用的技术分析方法、交易信号生成逻辑和评分体系。

---

## 一、技术指标计算

### 1.1 移动平均线 (MA)

计算多个周期的简单移动平均线，用于判断趋势和支撑/压力位。

| 周期 | 用途 |
|------|------|
| MA5 | 短期趋势，敏感度高 |
| MA10 | 短中期趋势 |
| MA20 | 中期趋势，常用支撑/压力位 |
| MA60 | 长期趋势，季度线 |

**计算公式：**
```
MA(n) = (P1 + P2 + ... + Pn) / n
```

### 1.2 相对强弱指标 (RSI)

衡量价格变动的速度和幅度，判断超买超卖状态。

**参数：** 周期 = 14 天

**计算公式：**
```
RS = 平均涨幅 / 平均跌幅
RSI = 100 - (100 / (1 + RS))
```

**信号判断：**
| RSI 值 | 状态 | 信号 |
|--------|------|------|
| < 20 | 严重超卖 | 强烈看涨 |
| < 30 | 超卖 | 看涨 |
| 30-40 | 偏弱 | 轻微看涨 |
| 40-60 | 中性 | 观望 |
| 60-70 | 偏强 | 轻微看跌 |
| > 70 | 超买 | 看跌 |
| > 80 | 严重超买 | 强烈看跌 |

### 1.3 MACD 指标

通过快慢指数移动平均线的差值判断趋势变化。

**参数：**
- 快线周期：12
- 慢线周期：26
- 信号线周期：9

**计算公式：**
```
EMA_fast = 12日指数移动平均
EMA_slow = 26日指数移动平均
MACD = EMA_fast - EMA_slow
Signal = MACD的9日EMA
Histogram = MACD - Signal
```

**信号判断：**
| 条件 | 信号 | 强度 |
|------|------|------|
| MACD上穿Signal（金叉） | 看涨 | 强 |
| MACD下穿Signal（死叉） | 看跌 | 强 |
| MACD > 0 且 柱状图 > 0 | 看涨 | 中 |
| MACD < 0 且 柱状图 < 0 | 看跌 | 中 |

### 1.4 布林带 (Bollinger Bands)

衡量价格波动范围，判断超买超卖和波动性。

**参数：**
- 周期：20 天
- 标准差倍数：2.0

**计算公式：**
```
中轨 = 20日MA
标准差 = 20日收盘价标准差
上轨 = 中轨 + 2 × 标准差
下轨 = 中轨 - 2 × 标准差
```

**信号判断：**
| 价格位置 | 信号 |
|----------|------|
| 触及/跌破下轨 | 看涨（超卖） |
| 触及/突破上轨 | 看跌（超买） |
| 在中轨下方 | 轻微看涨 |
| 在中轨上方 | 轻微看跌 |

### 1.5 波动率

衡量价格波动程度，用于风险评估。

**计算公式：**
```
日收益率 = 当日收盘价 / 前日收盘价 - 1
年化波动率 = 日收益率标准差 × √252 × 100%
```

## 二、表现指标分析

### 2.1 收益率指标

| 指标 | 计算方法 |
|------|----------|
| 总收益率 | (期末价 / 期初价 - 1) × 100% |
| 年化收益率 | ((1 + 总收益率)^(252/交易天数) - 1) × 100% |

### 2.2 风险指标

| 指标 | 计算方法 | 说明 |
|------|----------|------|
| 年化波动率 | 日收益率标准差 × √252 | 衡量价格波动程度 |
| 最大回撤 | (最高点 - 最低点) / 最高点 | 最大亏损幅度 |

### 2.3 风险调整收益

**夏普比率：**
```
夏普比率 = (年化收益率 - 无风险利率) / 年化波动率
```

**参数：** 无风险利率 = 1.5%（中国央行一年期存款基准利率）

**解读：**
| 夏普比率 | 评价 |
|----------|------|
| < 0 | 差，收益低于无风险利率 |
| 0 - 1 | 一般 |
| 1 - 2 | 良好 |
| > 2 | 优秀 |

## 三、交易信号生成

### 3.1 综合评分体系

系统采用多指标加权评分机制，综合7个维度生成交易建议。

**指标权重（尾盘标准）：**

| 指标 | 权重 | 说明 |
|------|------|------|
| 均线交叉 | 18% | MA5/MA20 金叉死叉 |
| MACD | 18% | 动能指标 |
| RSI | 15% | 超买超卖 |
| 布林带 | 15% | 波动区间 |
| 趋势 | 15% | 价格趋势方向 |
| 成交量 | 12% | 量价配合 |
| 溢价率 | 7% | 净值偏离度 |

**早盘策略调整：**

早盘数据不完整，采用更保守的权重：

| 指标 | 早盘权重 | 说明 |
|------|----------|------|
| 均线交叉 | 22% | 提高（更可靠） |
| MACD | 22% | 提高 |
| RSI | 18% | 提高 |
| 布林带 | 18% | 提高 |
| 趋势 | 15% | 不变 |
| 成交量 | 3% | 降低（早盘成交量不稳定） |
| 溢价率 | 2% | 降低（盘中净值波动大） |

### 3.2 单项指标评分

每个指标输出 -1 到 +1 的标准化分数：

| 分数范围 | 含义 |
|----------|------|
| +1.0 | 强烈看涨信号 |
| +0.5 ~ +0.8 | 看涨信号 |
| +0.2 ~ +0.3 | 轻微看涨 |
| 0 | 中性 |
| -0.2 ~ -0.3 | 轻微看跌 |
| -0.5 ~ -0.8 | 看跌信号 |
| -1.0 | 强烈看跌信号 |

### 3.3 综合信号判断

**尾盘阈值（标准）：**

| 综合得分 | 信号类型 | 置信度上限 |
|----------|----------|------------|
| ≥ 0.6 | 强烈买入 | 95% |
| ≥ 0.2 | 买入 | 80% |
| -0.2 ~ 0.2 | 持有 | 50% |
| ≤ -0.2 | 卖出 | 80% |
| ≤ -0.6 | 强烈卖出 | 95% |

**早盘阈值（保守）：**

| 综合得分 | 信号类型 | 置信度上限 |
|----------|----------|------------|
| ≥ 0.7 | 强烈买入 | 90% |
| ≥ 0.3 | 买入 | 75% |
| -0.3 ~ 0.3 | 持有 | 50% |
| ≤ -0.3 | 卖出 | 75% |
| ≤ -0.7 | 强烈卖出 | 90% |

## 四、价格建议计算

### 4.1 ATR (平均真实波幅)

用于衡量价格波动幅度，作为止盈止损的基准。

**计算公式：**
```
真实波幅(TR) = max(
    当日最高 - 当日最低,
    |当日最高 - 昨日收盘|,
    |当日最低 - 昨日收盘|
)
ATR = 14日TR平均值
```

### 4.2 价格建议

**买入信号时：**
| 价格类型 | 计算方法 |
|----------|----------|
| 建议买入价 | 当前价格 |
| 止盈价 | 当前价格 + 2 × ATR |
| 止损价 | 当前价格 - 1 × ATR |

**持有信号时：**
| 价格类型 | 计算方法 |
|----------|----------|
| 建议买入价 | 当前价格 - 0.5 × ATR（等待回调） |
| 止盈价 | 当前价格 + 1.5 × ATR |
| 止损价 | 当前价格 - 1.2 × ATR |

**卖出信号时：**
- 不提供止盈/止损价（A股不能做空，卖出即清仓）

## 五、风险评估

### 5.1 风险等级划分

基于年化波动率评估风险等级：

| 年化波动率 | 风险等级 |
|------------|----------|
| < 20% | 低 |
| 20% - 30% | 中 |
| > 30% | 高 |

### 5.2 溢价率分析

ETF 的交易价格与净值(NAV)的偏离程度：

```
溢价率 = (交易价格 - 净值) / 净值 × 100%
```

| 溢价率 | 状态 | 信号 |
|--------|------|------|
| < -3% | 高折价 | 强烈看涨 |
| -3% ~ -1% | 折价 | 看涨 |
| -1% ~ +1% | 正常 | 中性 |
| +1% ~ +3% | 溢价 | 看跌 |
| > +3% | 高溢价 | 强烈看跌 |

## 六、成交量分析

### 6.1 量价关系

| 条件 | 信号 | 说明 |
|------|------|------|
| 放量上涨（量比 > 1.5） | 看涨 | 资金流入，上涨有支撑 |
| 放量下跌（量比 > 1.5） | 看跌 | 资金流出，下跌可能加速 |
| 缩量上涨（量比 < 0.7） | 中性偏弱 | 上涨后续乏力 |
| 缩量下跌 | 中性 | 抛压减轻 |

**量比计算：**
```
量比 = 近5日平均成交量 / 总平均成交量
```

## 七、持仓分析

### 7.1 集中度指标

| 指标 | 说明 |
|------|------|
| 前5大持仓权重 | 衡量头部集中度 |
| 前10大持仓权重 | 衡量整体集中度 |

### 7.2 支撑压力位

**计算方法：**
```
支撑位 = 近20日最低价的最小值
压力位 = 近20日最高价的最大值
```

## 八、报告输出

### 8.1 交易建议信号

| 信号类型 | 图标 | 建议操作 |
|----------|------|----------|
| 强烈买入 | 🚀 | 积极建仓 |
| 买入 | 📈 | 适量买入 |
| 持有 | ➡️ | 观望等待 |
| 卖出 | 📉 | 减仓 |
| 强烈卖出 | 💥 | 清仓 |

### 8.2 报告内容

每份报告包含：
1. 实时行情（价格、涨跌幅、成交量）
2. 交易建议（信号、置信度、风险等级）
3. 价格参考（建议买入价、止盈价、止损价）
4. 技术指标状态（各指标的看涨/看跌/中性）
5. 历史表现（收益率、波动率、夏普比率）
6. 分析依据（生成建议的具体原因）
